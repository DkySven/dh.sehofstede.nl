AUR make and install script

aur

The Linux distro I use is Arch Linux and a few packages I have installed, come
from the Arch User repository. In order to automate the proces after downloading:
moving the tarballs to my build directory, unpacking it, executing makepkg -s and
installing it, I wrote a script.

Download it, set it to executable and move it to /usr/local/bin. Don't forget to change the build and download directory variables!
